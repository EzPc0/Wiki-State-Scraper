## Installation

1. Clone this repository or download as zip.

```sh
git clone https://github.com/EzPc0/Wiki-State-Scraper
```

2. install node

3. run  
```sh
npm install something
```

4. run while in the directory 
```sh
node index.js
```

and enjoy cool state stats
